🛠️ Features (it uses graphql as api)
Server-Side Rendering: Powered by EJS for dynamic content delivery.

Authentication: Full Login/Signup flow with Bcrypt password hashing and JWT service integration.

Email Services: Built-in OTP (One-Time Password) system via Nodemailer for signup verification and password resets.

Cart Logic: Session-based shopping cart management.

Database: Document-oriented storage using MongoDB and Mongoose.
